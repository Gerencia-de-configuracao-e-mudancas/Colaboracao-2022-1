Lista de comandos git

git init

Inicializa um repositório git em um diretório

git add .

Adiciona os arquivos atuais ao próximo commit

git status

Verifica o status atual do repositório git

git commit -m “mensagem do commit”

Cria um novo commit com uma mensagem

git init

Inicializa um repositório git em um diretório

rm -rf .git

Apaga um repositório

git log

Exibe os registros log do commit.

git checkout -b nome da branch

Cria uma branch paralela a branch main
